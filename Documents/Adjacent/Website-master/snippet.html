Here is the main relationships between the P tags

  • The <p> tag is the opening tag. Which means,the start of the paragraoh.
  • The </p> tag is the closing. This means the end of the paragraoh.

    • Here is the relationship between the a tags

  • The <a> indicates the start of the reference to a link.
  • The </a> tag indicates the end of the reference to the link. Think of this as as like the period.

    • The attributes on the a tag are: href, id, class, and target. These are their values and its effect:

    • href: "https://google.com"
      • Points to google as the destination
    • id: "the-best-link"
      • used to point to a style in a style sheet
    • class: "banana"
      • This is used to pick out multiple things within the code to be changed
    • target: "_blank"
      • This is used to say where the link will open up.
